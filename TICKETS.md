Let's make a website that allows the user to keep track of reserach linked to a certain project. We'll be adding inspiration, sketches, research elements and notes, and technologie references. The platform allows for collaboration on the projects


[x] As a user, I can create a new account, with a username, my first name, last name, email and password
[x] As a user, I can log in using my username or email, and password.
[x] As a user, I can change my password and other information on a settings page
[x] As a user, I can log out.
[x] As a user, I can add a new project, giving it a name, client, and deadline
[x] As a user, I can go to a detail page of the project. I can see four sections to the project, one about research, one about inspiration, one about sketches and one about technologies.
[x] As a user, I can invite another user to collaborate on a project.


[x] As a user, under the Research section of a project, I can add a finding, containing an optional URL, optional document and a short richt text body. 
[x] As a user, under the Research section of a project, I can mark a finding as "vital", showing it in red on the list
[x] As a user, under the Research section of a project, I can see a list of findings
[x] As a user, under the Research section of a project, I can search for a finding using a keyword in the richt text body
[x] As a user, I can remove research findings from the project

[x] As a user, I can invite a collaborator on a project using their email address. 
[x] As a user, I can remove a collaborator from a project
[x] As a collaborator, I can remove myself from a project
[x] As a user, I can see who added which elements to the project.

[x] As a user, I can add images (png, gif, webp, jpeg, jpg) to the inspiration section of the project. 
[x] As a user, I can mark images as "vital"
[x] As a user, I can link image items to research items within the same project
[x] As a user, I can remove items from the inspiration section.


[x] As a user, I can add images (png,jpg) to the sketches section of the project. 
[x] As a user, I can mark sketches as "vital"
[x] As a user, I can remove items from the sketches section.
[x] Sketches are resized upon upload, to 200px for thumbnail and 1200px for display
[x] As a user, I can select a sketch and see the larger version when clicking on the image in an overlay


[ ] As a user, under the technology section of a project, I can add a technology, containing an URL, optional technical PDF document, a short richt text body, an amount necessary, and a price per unit. I can also mark if it is a rented or bought item. 
[ ] As a user, under the technology section of a project, I can mark a item, showing it in red on the list
[ ] As a user, under the technology section of a project, I can see a list of items, as a table
[ ] As a user, under the technology section of a project, I can search for an item using a keyword in the richt text body
[ ] As a user, I can remove research findings from the project
