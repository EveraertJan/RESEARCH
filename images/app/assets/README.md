# Assets Directory

Place your app assets here:

## Required Images

- `icon.png` - App icon (1024x1024 px)
- `splash.png` - Splash screen (1242x2436 px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Generate Assets

You can generate all required assets from a single icon using:

```bash
npx expo-optimize
```

Or use online tools:
- [App Icon Generator](https://appicon.co/)
- [Expo Icon Generator](https://buildicon.netlify.app/)

## Temporary Placeholders

For development, you can use solid color placeholders:
- Create 1024x1024 PNG with your brand color
- Save as `icon.png`, `adaptive-icon.png`
- Create 1242x2436 PNG for splash screen
